# Changelog

All notable changes to Hasalty will be documented in this file.

## [1.0.0] - 2026-02-09

### Added
- **Authentication System**
  - Secure login with role-based access (Parent/Child)
  - User registration with validation
  - Password requirements (8+ characters, starts with capital letter)
  - Username validation (must start with "parent" or "child")

- **Dashboard & Home Screen**
  - Card balance display with active status
  - Weekly spending summary
  - Mini bar chart for spending trends
  - Recent activity feed with transaction history
  - Financial goals progress tracking
  - Action buttons (Allowance, Top Up, Freeze)
  - Dark mode support

- **Analytics & Reporting**
  - Weekly, monthly, and yearly spending views
  - Interactive donut charts for spending breakdown
  - Category-based spending analysis
  - Spending trends and comparisons
  - AI-powered recommendations
  - Saving star alerts and limit warnings

- **Card Management**
  - Virtual card display with balance
  - Card freeze functionality
  - Top-up capabilities
  - Card status indicators
  - Card-specific features and controls

- **Financial Tracking**
  - Real-time spending categorization
  - Transaction history and details
  - Multiple spending categories (Gaming, Food & Drink, Travel, Gadgets)
  - Spending alerts and notifications
  - Weekly spending limits

- **Multi-Platform Support**
  - iOS (iOS 11.0+)
  - Android (Android 5.0+)
  - Web (Modern browsers)
  - Windows (Windows 10+)
  - macOS (macOS 10.13+)
  - Linux (Ubuntu 18.04+)

- **UI/UX Features**
  - Material Design 3 compliance
  - Responsive design for all screen sizes
  - Dark/Light mode toggle
  - Smooth navigation between screens
  - Professional color schemes and typography

### Technical Details
- Built with Flutter 3.3.0+
- Dart 3.3.0+
- fl_chart for advanced analytics visualization
- ~1,747 lines of production code

### Notes
- Beta release - production-ready features
- Future backend integration planned
- Security enhancements in development roadmap
